# Jogo Vida de Estudante

O objetivo é sobreviver aos adversários (animais) que aparecem randomicamente nas salas do jogo, coletando e comprando itens para melhorar o poder de ataque e defesa do personagem. Além disso, é necessário coletar 4 chaves que abrirão o elevador, sendo que estas serão dadas ao vencer os "chefões" do jogo, representados pelos professores presentes em algumas salas.
Existem 2 salas secretas, que contêm itens especiais gratuitamente.

O executável do jogo se encontra na pasta dist.

Ao executar o jogo, será possível escolher o modo solo e modo online, sendo que pro segundo é necessário o servidor (presente no outro repositório) estar online.
